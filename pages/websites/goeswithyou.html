MailChimp Goes with You

Design: David Sizemore
Art Direction: Marco Suarez
Video: Troy Stains
Photo: Jason Travis

MailChimp Goes with You was a campaign highlighting our range of mobile apps, illustrating how using MailChimp doesn't mean you have to be sitting behind a desk. Marco Suarez headed up the project; I was responsible for designing the page.

View the Site

I like mocking up my work when it has interative elements, so we can talk through them before time is spent building them out. In this case, that meant changing the 'goes with you' script from a weird, parallax-y element to a page header that literally goes with you down the page. See what we did there? Cause it goes with—okay, you've got it.

I wrote a bit about that script, and how James Victore came to write it, on the DesignLab blog.
